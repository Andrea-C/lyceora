# Lyceora M2.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Roles (Better Auth admin plugin) + impersonation, app-wide navigation, admin dashboard with per-profile drill-in, and the tester-reported UX fixes, per `docs/superpowers/specs/2026-07-16-lyceora-m2.1-usability-design.md`.

**Architecture:** Role lives on the Better Auth user (plugin-managed); every admin surface is server-gated by a `requireAdmin*` helper on top of plugin checks. Navigation is one server component in a new `/app` segment layout. The parent page's report sections extract into a shared `ProfileReport` consumed by both parent and admin drill-in.

**Tech Stack:** Next.js 16 App Router, Better Auth ^1.2 (`admin` plugin), Drizzle/PGlite, next-intl, vitest + Playwright.

- [ ] **Step 4: Layout** `apps/web/src/app/[locale]/app/layout.tsx`:

```tsx
import { cookies, headers } from "next/headers";
import { redirect } from "next/navigation";
import { auth } from "@/lib/auth";
import { isAdmin } from "@/lib/session";
import { db } from "@/lib/db";
import * as repo from "@/server/repo";
import { AppNav } from "@/components/AppNav";

export default async function AppLayout({ children, params }: {
  children: React.ReactNode; params: Promise<{ locale: string }>;
}) {
  const { locale } = await params;
  const session = await auth.api.getSession({ headers: await headers() });
  if (!session) redirect(`/${locale}`);

  let activeProfileName: string | null = null;
  const profileId = (await cookies()).get("lyceora_profile")?.value;
  if (profileId) {
    try {
      activeProfileName = (await repo.getOwnedProfile(db, session.user.id, profileId)).displayName;
    } catch { /* stale/foreign cookie: nav simply omits the chip; pages keep their own redirects */ }
  }

  async function logout() {
    "use server";
    await auth.api.signOut({ headers: await headers() });
    (await cookies()).delete("lyceora_profile");
    redirect(`/${locale}`);
  }

  return (
    <>
      <AppNav locale={locale} isAdminUser={isAdmin(session.user)} activeProfileName={activeProfileName} logoutAction={logout} />
      {children}
    </>
  );
}
```
